添加远程仓库：step1: git remote add <远程仓库别名> <远程仓库地址>
	       step2: git branch -M <分枝名> 自己创建当前分枝的名字
	       step3: git push -u<远程仓库名><分枝名>
查看远程仓库：git remote -v
拉取远程仓库内容：git pull <远程仓库名><远程仓库分支名>:<本地分枝名>
                                                                ---------------------------------
				若相同则可以省略冒号后面部分
